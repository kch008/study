# Shifter

### 1. 정의 
- 2진수의 각 자리를 왼쪽 또는 오른쪽으로 이동해주는 회로

### 2. 논리적 시프트 (logical shift)
- bit를 왼쪽 또는 오른쪽으로 이동시키고 생기는 빈 공간을 0으로 채운다.

#### 예시
- 11001 >> 2 = 00110\
  11001 << 2 = 00100

### 3. 산술적 시프트
- logical shift와 동일하나, right shift에서만 생기는 빈 공간을 msb로 채운다.

#### 예시 
- 11001 >>> 2 = 11110\
  11001 <<< 2 = 00100


### 4.순환 시프트(circular shift) 
- 회전(rotate)이라고도 부르며, 
  bit를 원형으로 순환시킨다. 즉, 밀려 나온 bit들은 반대쪽 끝으로 들어간다.

#### 예시 
- 11001 ROR 2 = 01110\
  11001 ROL 2 = 00111
