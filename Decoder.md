# Encoder + Decoder

### 1. 정의
>인코더가 데이터를 내부 표현(벡터)으로 압축 디코더가 그 내부 표현을 바탕으로 출력 데이터를 생성
	![Definition.](./Images/Encoder+Decoder/Definition.png) 

### 2. 필요성
>-- 입력과 출력이 형태가 다를 때
-- 입력과 출력의 길이가 다를 때
-- 입력을 요약하고, 의미 중심으로 처리해야 할 때

##### ex.
- 영어 → 프랑스어 번역
- 긴 문장을 짧게 요약
- 이미지 → 캡션 생성
- 음성 → 텍스트 변환
